# Server README

Open a shell in the git root folder then :
Run "java -jar spreadit-server\spreadit-server-0.1.jar"
Wait for "=== SpreadIt server started ===" to appear
Type enter

It will log request received and what it send to GCM.
Sometimes the windows shell blocks server, type enter to unblock it.

You can run curl tests available into src\main\resources\curl-tests.txt
